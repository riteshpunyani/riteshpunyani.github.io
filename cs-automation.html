Case Study 01
🚀 Automating PAN Verification for KYC Compliance
📌 Background & Challenge
As part of a KYC compliance project, our team had to validate customer PAN details using the Income Tax Department's portal. For a sample size of 10,000+ PANs, the traditional manual process was extremely time-consuming, error-prone, and consuming reviewer effort on repetitive tasks — one by one.
⚙️ Solution
1
Recorded the manual verification steps once on the Income Tax portal using macro software
2
Linked the process with Excel, where all 10,000+ PAN numbers were listed in a structured sheet
3
Built a workflow that automatically entered each PAN, captured name and date of birth, and recorded results systematically
4
Executed for 10,000+ cases — completing in hours what would have taken several days

Case Study 02
🚀 Automating SAP SoD Script Updates
📌 Background & Challenge
As part of the annual compliance cycle, the team updated Segregation of Duties (SoD) testing scripts within SAP — each spanning 250–300 pages, requiring changes to user IDs and access rights. The manual process consumed 20–25 man-days of effort annually, with high risk of copy-paste errors tying up skilled resources in low-value tasks.
⚙️ Solution
1
Analysed the structure of the SAP SoD testing scripts to identify all updatable fields
2
Built a VBA macro to rewrite and update the scripts automatically based on new user rights data from Excel
3
Ran the macro — generating fully updated 250–300 page scripts in minutes instead of weeks
✅ Results & Recognition

Case Study 03
🚀 Automated File Renaming for Litigation Review
📌 Background & Solution
During a litigation-related investigation, the team had to rename hundreds of documents based on a specific sequencing provided in a spreadsheet — to make referencing easy during review. Doing this manually had previously taken 2–3 days.

Using VBA, I wrote a script that instantly renamed all files per the sequencing, with a smart workaround to retrieve filenames even when not listed in the spreadsheet. Total coding effort: 15–20 minutes. Total execution time: 10 seconds.
